## Sobre el Proyecto

Este portafolio fue diseñado con un enfoque minimalista, limpio y profesional.

### Objetivos principales

* Mostrar mi experiencia laboral.
* Presentar mis proyectos destacados.
* Compartir mis habilidades técnicas.
* Facilitar el contacto profesional.
* Mantener una interfaz adaptable (responsive) para dispositivos móviles y escritorio.

---

## Tecnologías Utilizadas

* React
* Astro
* TailwindCSS
* Vercel

## Instalación y Ejecución

Si deseas ejecutar el proyecto de manera local:

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/tu-repositorio.git

# Entrar al proyecto
cd tu-repositorio

# Instalar dependencias
bun install

# Ejecutar en desarrollo
bun run dev
```
